# Event Management and Photography Website

## Project Description
This is a simple Event Management and Photography website created using HTML, CSS, and JavaScript.

The website allows users to:
- View event services
- See event photo gallery
- Read customer reviews
- Contact the event company

## Features
- Home Page
- Services Section
- Event Gallery
- Customer Reviews
- Contact Information (Phone and Gmail)

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Use
1. Download the project files.
2. Open the file "index.html" in any web browser.
3. Navigate through the website using the menu.
